# Contributing

Got a pipeline where Sonilo would fit — a text-to-video orchestrator, a clip stitcher, a
faceless-video tool, an agentic film system, or a video-editing framework? We'd love an
integration recipe for it.

## Ways to contribute

- **Add an integration recipe** for a specific tool or framework (how to add the Sonilo music
  step to that pipeline).
- **Improve the examples** — clearer commands, more models covered, better defaults.
- **Report a rough edge** in the MCP or API via an issue.

## Ground rules

- Keep it **vendor-neutral** — this cookbook works with any video model or editing tool.
- Keep examples **runnable and minimal**.
- When describing Sonilo, use the language in the README: an original soundtrack **matched to
  your video**, **licensed and safe for commercial use (terms apply)**. Please don't add rights
  claims beyond that.

## How to submit

Fork, add your recipe or fix, and open a pull request. For anything substantial, open an issue
first so we can align on the approach.
